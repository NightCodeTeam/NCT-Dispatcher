# Night Code Team Dispatcher

Это система для отслеживания и работой со сбоями, багами и ошибками в работе других серсисов. Проект вдохновлен Netflix Dispatcher однако на данный момент функционал сильно ограничен в возможностях, но занимает памяти ресурсов в разы меньше.
Главное приложение - app, создает API и управляет ботов в Telegram. Dispather_Maker - базовый пакет для создания запросов, его не обзяательно использовать, однако у него есть несколько удобных методов.

## V1

На данный момент актуальная версия 1 с базовым функционалом.

## Dispatcher API

API представляет лишь одну точку - post incident. Во время этого запроса приложение проверяет код подключения к api для защиты от стороних запросов, верифицирует приложение и записывает в базу данных полученный инцидент.

API не будет отзываться если в запросе не будет корректного кода доступа и правильно указанного имени приложения.

После записи инцидента в Telegram чат отправляется сообщение с полным описанием инцидента. В первой версии проекта чат считается административным и каждый пользователь чата может закрыть или удалить инцидент а так же имеет полный доступ к приложению.

Так же используя команду /start откроется меню где пользователь может открыть новые не закрытые инциденты, посмотреть все инциденты, подробную информацию обо всех приложениях и сопутствующих инцидентах.



Дополнительной командой является /new которая создает новое приложение в базу и создает ему код доступа, который можно скопировать.

## Dispatcher Maker

Это пакет для вашего приложения. Содержит 4 основных компонента:

- Dispatcher - класс для отправки инцидента в синхронном режиме
- DispatcherAsync - класс для отправки инцидента в ассинхронном режиме
- post_incident - функция отправки инцидента в синхронном режиме
- post_incident_async - функция отправки инцидента в ассинхронном режиме
